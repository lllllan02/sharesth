## Share Something

### 项目简介

Share Something 是一个用于分享文件和信息的平台，旨在提供简单快捷的分享体验。

### 安装指南

1. 克隆仓库：`git clone <repository-url>`
2. 进入项目目录：`cd sharesth`
3. 安装依赖：`go mod tidy`
4. 运行项目：`go run main.go`

### 使用说明

- 访问 `http://localhost:8080` 以使用该平台。
- 上传文件或信息，并生成分享链接。
