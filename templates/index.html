Markdown 小技巧

  • 使用 # 创建标题 (# 一级标题, ## 二级标题)
  • 使用 **文本** 或 __文本__ 来加粗文本
  • 使用 *文本* 或 _文本_ 来斜体显示
  • 使用 - 或 * 创建无序列表
  • 使用 1. 2. 3. 创建有序列表
  • 插入链接: [链接文本](URL)
  • 插入图片: ![替代文本](图片URL)
  • 创建表格: | 表头 | 表头 | 然后下一行 | --- | --- |
